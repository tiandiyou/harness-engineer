# 特权账号系统 E2E 测试框架

> 基于 Harness Engineering 框架的多Agent端到端测试系统

## 🎯 框架目标

本框架用于**特权账号管理系统**的端到端自动化测试，通过多Agent协作实现：
- 自动化生成测试用例
- 自动执行测试
- 自动分析结果并生成报告

## 🚀 快速开始

### 1. 环境要求

- Python 3.8+
- pytest
- requests
- PyYAML

### 2. 安装依赖

```bash
pip install pytest requests pyyaml
```

### 3. 配置测试环境

编辑 `config/test_config.yaml` 配置：
- 测试服务器地址
- 数据库连接
- 测试账号

### 4. 运行测试

```bash
# 方式1: 使用协调器运行（完整流程）
python run_tests.py

# 方式2: 直接使用pytest运行
pytest tests/e2e/tests/ -v
```

### 5. 查看报告

测试执行后会自动生成报告，包含：
- 测试摘要
- 通过率统计
- 失败分析
- 改进建议
